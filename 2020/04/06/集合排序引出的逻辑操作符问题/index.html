发表于 2020-04-06 | 分类于 freeCodeCamp | | 阅读数
字数统计: 1.1k | 阅读时长 ≈ 4分钟

我们需要写一个函数，它接收两个或多个数组为参数。我们需要对这些数组中所有元素进行去除重复元素的处理，并以数组的形式返回去重结果，结果数组中的元素顺序必须与其传入的顺序保持一致。
例如：
([1, 3, 2], [5, 2, 1, 4], [2, 1])应该返回[1, 3, 2, 5, 4]，
([1, 3, 2], [1, [5]], [2, [4]])应该返回[1, 3, 2, [5], [4]]，
([1, 2, 3], [5, 2, 1])应该返回[1, 2, 3, 5]，
([1, 2, 3], [5, 2, 1, 4], [2, 1], [6, 7, 8])应该返回[1, 2, 3, 5, 4, 6, 7, 8]

解决方法：spread操作符取出传入的参数中的每一个数组，将这些数组concat成一个数组，然后对这一个数组使用reduce方法去重，去重原理：reduce基本使用
